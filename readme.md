# Modular Student Management System

A modular student management system built with Python using Object-Oriented Programming (OOP).

This project allows users to:

- Add students
- Search students
- Delete students
- Add courses
- Search courses
- Delete courses
- Enroll students in courses
- Unenroll students from courses

---

# Features

## Student Management
- Add new students
- Prevent duplicate students
- Search students by:
  - ID
  - Name
- Delete students

## Course Management
- Add new courses
- Prevent duplicate courses
- Search courses by:
  - ID
  - Name
- Delete courses

## Enrollment System
- Enroll students into courses
- Prevent duplicate enrollments
- Unenroll students from courses
